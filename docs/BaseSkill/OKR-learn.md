# OKR 理论学习


## 定义

OKR，即目标（object）和关键结果（key result）。其有如下4个特点：

- 目标清晰：即指定OKR之前必须要想清楚这样做的价值。
- 动态周期：即按照业务节奏来制定OKR周期，不必固定。
- 公开透明：即部门内部对每个人可见。
- 精力聚焦：即一个O的KR一般在3-5条，把主要精力集中在关键结果的达成上，且一般KR定量描述。

在实际应用中，一般OKR会有三层结构，即一个O对应多个KR，一个KR对应多个Action，其是企业的使命-愿景-战略链路的缩影。


## 与传统管理方式的核心区别

传统的管理模式（如KPI），属于评估导向，一般分为目标制定-目标实施-目标评价的闭环。在此过程中，由于两者耦合度较高，往往存在压低目标，夸大完成度的情况。

OKR则属于发展导向，其针对传统管理模式的问题，将上述过程进行拆分，实现了目标管理与评价管理分离的效果。价值创造在目标制定和目标辅导环节完成，而价值评价在后续的绩效评价和沟通环节完成。此时，后半部分的目标完成情况只用作改进，不用作奖励和惩罚。同时，OKR保持公开透明，对团队其他人可见。


## 制定

OKR强调自下而上的目标制定，旨在让员工发挥一定的主动性，帮助组织寻找突破方向。一般来说，OKR的指定遵循如下步骤：

- 团队OKR制定，即团队成员头脑风暴，共同指定并讨论出团队OKR，明确团队未来的发展方向。
- 个人OKR制定，以团队OKR作为参照大方向，输出个人OKR，明确个人的努力方向。
- OKR微调：针对公开意见以及实际情况进行微调。
